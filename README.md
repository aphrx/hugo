This is Hugo, a customizable Slack bot built with FastAPI and Slack Bolt.

## Quick Start (Local)

1. Clone the repo  
2. Create a `.env` file (copy from `.env.example`)
3. Install dependencies:

```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
